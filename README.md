## `vt-plugin create`

Create a new plugin

```
USAGE
  $ vt-plugin create 

OPTIONS
  -h, --help               show CLI help
  -n, --name=name          Internal Name of the plugin
  -p, --display-name=name  Public Name of the plugin
  -d, --description        Description of the plugin
  
```
